# RustShortner

Inspired by platforms like [LinkedIn](https://www.linkedin.com), [Bitly](https://bitly.com), and other link shorteners.

## Features

- **Shorten links without login**
- **Shorten links with login**:
  - View analytics
  - Generate and download QR codes
  - Manage your account
- **Customizable short links**
- **Secure and reliable**
- **Mobile-friendly interface**
- **Future services planned**
- **Ongoing bug fixes and improvements**

> RustShortner aims to provide a fast, secure, and user-friendly experience for all your link management needs.
